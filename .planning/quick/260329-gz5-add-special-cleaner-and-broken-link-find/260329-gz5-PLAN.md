**2. Create backend/app/routers/tools.py:**
- Follow same patterns as agent.py router (imports, get_redis dependency, etc.)
- `router = APIRouter(prefix="/tools", tags=["tools"])`
- Use `REDIS_URL = os.getenv("REDIS_URL", "redis://redis:6379/0")`
- Copy the `get_redis` async generator from agent.py.

**POST /tools/special-cleaner/run**
- Pydantic body: `class SpecialCleanerRequest(BaseModel): dry_run: bool = False; min_size_mb: int = 75`
- Build command string: `python /app/scripts/special_cleaner.py`
  - If dry_run: append ` --dry-run`
  - Always append ` --min-size-mb {min_size_mb}`
- Generate `run_id = str(uuid.uuid4())`
- `await r.lpush("agent_commands", json.dumps({"command": cmd, "macro_name": "special-cleaner", "run_id": run_id, "is_last": True}))`
- Return `{"run_id": run_id}`

**POST /tools/broken-link-finder/scan**
- No request body needed
- Generate `run_id = str(uuid.uuid4())`
- Create a NEW redis connection (not from dependency — need to manage pubsub lifecycle manually): `r_sub = redis.from_url(REDIS_URL)`
- `pubsub = r_sub.pubsub()` and `await pubsub.subscribe("agent_responses")`
- THEN publish command via the injected `r` dependency: `await r.lpush("agent_commands", json.dumps({"command": "python /app/scripts/broken_link_finder.py --json", "macro_name": "broken-link-finder", "run_id": run_id, "is_last": True}))`
- Collect results in a loop with 300s timeout using `asyncio.wait_for`:
  - `async for message in pubsub.listen()`: skip if type != "message"
  - Parse JSON, skip if `data.get("run_id") != run_id`
  - If `data.get("status") == "streaming"`: try `json_mod.loads(data.get("message", ""))` — if valid dict with "path" key, append to results list
  - If `data.get("status") in ("completed", "error")` or `data.get("is_last") == True` with status in completed/error: break
- In finally block: `await pubsub.unsubscribe("agent_responses")` and `await r_sub.aclose()`
- Return `{"files": results}` (list of `{path, status, msg}` dicts)

**POST /tools/broken-link-finder/delete**
- Pydantic body: `class DeleteRequest(BaseModel): paths: list[str]`
- `import shlex`
- Build command: `"rm -f " + " ".join(shlex.quote(p) for p in body.paths)`
- Generate run_id, lpush to agent_commands with macro_name "broken-link-finder-delete", is_last True
- Return `{"run_id": run_id}`

**3. Update backend/app/main.py:**
